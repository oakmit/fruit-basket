1. Ensure that solution.exe and desired fruit basket files are in the same location. Open 'solution.exe.'
2. You will be prompted for a file name. Type in the full name of the file, i.e. 'basket.csv'
3. Data requested regarding the composition of the fruit basket will be displayed. You will be asked if you want to keep going.
	- If Yes, type in "y" or "Y" --> You will be prompted for another file name
	- If No, typing in anything else will close the program
